A · MV3 分批段落翻译（推荐）

使用 content script 提取段落，service worker 调用本地 OpenAI 兼容接口，逐批回填页面。

优点

  • 体验接近沉浸式翻译
  • 本地接口不受网页 CORS 限制
  • 可显示翻译进度

限制

  • 需处理动态网页
  • 首次翻译大页面需要等待

B · 网页脚本直接请求

结构较少，但网页安全策略和本地接口跨域规则更容易导致失败。

优点

  • 代码更少

限制

  • CORS 风险
  • 网页可干扰执行环境

C · 侧边栏全文翻译

不修改网页 DOM，但失去原文段落旁边直接阅读译文的沉浸式体验。

限制

  • 对照阅读不够直观
